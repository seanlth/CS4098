## Installation
```
git clone https://github.com/seanlth/CS4098.git
cd CS4098
make install
```
## Starting

```
make run
```

Open ```0.0.0.0:8000``` in a browser for PML roundtrip
Open ```0.0.0.0:8000/ace``` in a browser for code editor prototype
